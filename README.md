# System Design Of 3U Precision Agriculture CubeSat

Here, the system design of a precision agricultural cube is presented.

The project is defined in 5 phases, as follows:

1- Phase 0(SSD-Project-Phase0): In this phase, the mission definition is done.

2- Phase A(PROJ-SYS-RPT-002): In this phase, the mission requirements are extracted and the initial concept of operations and initial sizing are done.

3- Phase B1(PROJ-SYS-RPT-003): In this phase, the requirements are converted from the mission requirements level to the system requirements level and presented using the basic concepts of model based systems engineering, satellite function and physics.

4- Phase B2(PROJ-SYS-RPT-004): In this phase, the satellite subsystems are selected and their sizing is done. And according to the sizing, the satellite computer-aided design is done at level 1.

5- Phase C(PROJ-SYS-RPT-005): In this phase, numerical analyses are done.
